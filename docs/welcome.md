# Welcome to Essential
<h4 class="welcome-subtitle">Essential is the first declarative blockchain, advancing the shift from value extraction to intent satisfaction.</h4>

Essential’s declarative model enables users to submit intents—high-level goals—while the network determines the optimal way to fulfill them. This shift brings precision and simplicity to Web3, enabling:

- **For Users**: Predictable, seamless outcomes.
- **For Developers**: Reduced complexity and faster innovation.
- **For Web3**: Scalability, lower costs, and decentralization.

By adopting a fully declarative approach, Essential aligns blockchain capabilities with user intent, unlocking Web3’s true potential.
